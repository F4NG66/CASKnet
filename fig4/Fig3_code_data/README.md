# Fig3 final customer-request package

This package follows the customer's original request exactly:
- remove the blue method throughout the figure;
- split the green and grey bars into separate bars in panels a and b;
- keep the figure publication-style and readable;
- do not change the source data.

How to run:
1. Install Python 3.10+.
2. Install dependencies: `pip install -r requirements.txt`
3. Run: `python plot_fig3.py`

Outputs are written to the `outputs/` folder.
